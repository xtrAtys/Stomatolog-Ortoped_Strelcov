# Ortoped_Strelcov
# Сайт-визитка врача стоматолога-ортопеда (Stomatolog-Ortoped Strelcov)

## 📋 О проекте

**Stomatolog-Ortoped Strelcov** — это статичный веб-сайт, выполняющий функцию персональной цифровой визитной карточки для врача стоматолога-ортопеда. Проект представляет собой лаконичный и информативный одностраничный сайт, созданный для быстрого знакомства потенциальных пациентов с доктором и его услугами.

Сайт разработан на чистом HTML с использованием CSS и JavaScript и не требует сложной серверной инфраструктуры, что обеспечивает его высокую скорость загрузки и простоту размещения в интернете.

## ✨ Функциональность и особенности

*   **Основная информация о специалисте:** Представление врача, его квалификация и специализация (стоматолог-ортопед).
*   **Описание услуг:** Краткий обзор предоставляемых стоматологических услуг в области ортопедии.
*   **Контактные данные:** Удобное отображение информации для связи (телефон, адрес клиники, возможно, ссылки на мессенджеры).
*   **Визуальная привлекательность:** Использование современного веб-дизайна для создания профессионального имиджа.
*   **Адаптивная верстка:** Сайт корректно отображается на различных устройствах (компьютерах, планшетах, смартфонах), что обеспечивается использованием CSS.

## 🛠 Технологии

*   **HTML5** — базовая структура всех страниц сайта (100% кода).
*   **CSS3** — стилизация элементов, создание привлекательного внешнего вида и адаптация под разные экраны.
*   **JavaScript** — добавление интерактивных элементов (например, мобильное меню, плавная прокрутка).

## 🚀 Как открыть сайт

Проект является полностью статичным, поэтому его можно открыть несколькими способами:

### Способ 1: Прямой доступ через GitHub Pages
Если владелец репозитория включил GitHub Pages, сайт доступен по адресу:
`https://xtrAtys.github.io/Stomatolog-Ortoped_Strelcov/`
(или в ветке `gh-pages`).

### Способ 2: Локальный запуск (на вашем компьютере)
1.  **Клонируйте репозиторий** (или скачайте его как ZIP и распакуйте):
    ```bash
    git clone https://github.com/xtrAtys/Stomatolog-Ortoped_Strelcov.git
    ```
2. Перейдите в папку с проектом:

3. Откройте файл index.html в любом современном браузере (Chrome, Firefox, Safari, Edge). Для этого можно просто дважды кликнуть по файлу или перетащить его в окно браузера.

## 📁 Структура проекта
index.html - Главная (и единственная) страница сайта.

css/ - Папка, содержащая файлы стилей (.css).

js/ - Папка с JavaScript-файлами (.js), отвечающими за интерактивность.

icons/ - Папка для иконок и изображений, используемых на сайте.

.idea/ - Служебная папка настроек для IDE (например, PyCharm) — не влияет на работу сайта.
